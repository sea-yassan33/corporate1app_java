# 概要
* Javaの学習練習としてコーポレートサイトのサンプルを作成
* Figmaでデザインおよびフロント設計
* Javaとspring_bootでバックエンド設計

# 開発環境

### フロントエンド
- HTML
- CSS
- JavaScript
- jQuery(flame-work)
- slick(flame-work)

### バックエンド
- Java
- Spring Boot3(flame-work)

### データベース
- MySQL

### 総合開発環境
- IntelliJ IDEA
- Visual Studio Code

### デザイン設計開発ツール
- Figma
- Canva
